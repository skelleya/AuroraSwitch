# Hybrid Software KVM Overview

- **Architecture direction**: Hybrid software KVM with USB-over-IP for HID redirection plus HDMI/USB-C capture for video/audio ingestion.
- **Host platform**: Windows 11.
- **Implementation stack**: `C#` with WPF for the desktop UI and host control services.
- **Target endpoints**: Primary host PC, HP laptop over HDMI/USB-C, and MacBook Air M2 with multi-monitor ambitions.
- **Key goals**: Reliable hotkey-based switching, exclusive peripheral control, low-latency interaction, and extensible support for additional machines.
